### 🎖️ 프로젝트 목표 및 기대효과

#### 목표
파인튜닝 적용한 sLLM 을 활용해서 Agent 형 RAG Application 을 구축
직원들의 정보검색시간을 단축하여 업무 효율성 향상을 기대
#### 기대효과
서비스 구축을 통해 신속한 업무정보 제공
회사의 업무 효율성 향상 및 시간 절약
자원 관리를 최적화하고, 회사의 생산성을 높이는 데 중요한 역할
